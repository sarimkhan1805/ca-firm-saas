# 🏦 CA Firm SaaS

A production-ready **Chartered Accountancy Firm Management System** built with **React, Node.js/Express, and PostgreSQL** — designed to streamline client management, task tracking, and reporting for CA firms.

## 📌 Features

- 🔐 **User Authentication** — Secure login & signup with role-based access
- 👥 **Client Management** — Add, update, and manage client profiles and records
- ✅ **Task Management** — Assign and track tasks across team members
- 📊 **Dashboard & Reports** — Visual overview of firm activity and performance
- 🛡️ **Role-Based Access Control** — Admin and staff permission levels

---

## 🛠 Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **Authentication:** JWT + Cookies
- **Deployment:** Railway *(coming soon)*

## ⚙️ Environment Variables

Create a `.env` file inside the `backend/` folder:

```
DATABASE_URL=your_postgresql_connection_string
JWT_SECRET=your_secret_key
PORT=5000
NODE_ENV=development
```

For production (Railway), configure:

- `DATABASE_URL`
- `JWT_SECRET`
- `NODE_ENV=production`

## 🧪 Setup & Run Locally

### 1. Clone the repository

```bash
git clone https://github.com/sarimkhan1805/ca-firm-saas.git
cd ca-firm-saas
```

### 2. Install backend dependencies

```bash
cd backend
npm install
```

### 3. Install frontend dependencies

```bash
cd ../frontend
npm install
```

### 4. Start the backend server

```bash
cd ../backend
npm start
```

### 5. Start the frontend

```bash
cd ../frontend
npm start
```

### 6. Open in browser

```
http://localhost:3000
```
